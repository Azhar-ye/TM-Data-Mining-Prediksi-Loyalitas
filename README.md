# Prediksi Loyalitas Pelanggan Berdasarkan Data Perilaku Belanja

Repository ini berisi Tugas Mandiri (TM) mata kuliah Fundamentals of Data Mining yang membahas
penerapan algoritma klasifikasi untuk memprediksi loyalitas pelanggan berdasarkan
data perilaku belanja pelanggan.

## Dataset
Dataset yang digunakan merupakan dataset perilaku belanja pelanggan yang terdiri
dari 3900 data dengan 16 atribut, mencakup informasi demografis, transaksi, dan
kebiasaan belanja pelanggan.

## Metode
Algoritma data mining yang digunakan dalam tugas ini adalah:
- C4.5 (Decision Tree)
- K-Nearest Neighbor (KNN)
- Random Forest

## Tools
- Python
- Google Colab
- Scikit-learn

## Hasil
Berdasarkan hasil evaluasi model, algoritma Random Forest menghasilkan performa
terbaik dengan nilai akurasi sekitar 80% dalam memprediksi loyalitas pelanggan.
